# TPDv_code
Code for paper 'Transformed primal-dual methods with variable preconditioners'

Files in this folder are based on iFEM package by Long Chen (University of California, Irvine).

To run the code in this folder, first download the repository at https://github.com/lyc102/ifem

Main file for DF problem is Darcy_ForchheimerP0P1_TPD.m

For nonlinear Maxwell problem, files begin with cubeMaxwellSaddleNonlinearSOLVER.m
